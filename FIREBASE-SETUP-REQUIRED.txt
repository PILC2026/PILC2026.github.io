=====================================================
WHY IS THIS NEEDED?
=====================================================

Firebase Firestore has security rules that control who can read/write data.
By default, it blocks all access for security.

We need to allow:
- Anyone to register (create user documents)
- Users to login (read user documents)
- Admins to approve users (update user documents)
- Admins to list all users

The rules in "firestore-rules.txt" provide these permissions.
